# DVWA Web Application Penetration Testing

## Project Overview
A complete web application penetration test conducted 
against the Damn Vulnerable Web Application (DVWA) 
running on Kali Linux. This project demonstrates 
real-world web vulnerability identification, 
exploitation, and documentation.

## Vulnerabilities Identified
| # | Vulnerability | Severity | CVSS |
|---|--------------|----------|------|
| 1 | SQL Injection | HIGH | 8.8 |
| 2 | Reflected XSS | MEDIUM | 6.1 |
| 3 | Stored XSS | HIGH | 8.0 |
| 4 | CSRF | MEDIUM | 6.5 |

## Tools Used
- Kali Linux
- DVWA
- Firefox Browser
- Burp Suite
- CrackStation

## Environment
- Attacker: Kali Linux
- Target: DVWA (127.0.0.1)
- Web Server: Apache 2.4.67
- Database: MySQL
